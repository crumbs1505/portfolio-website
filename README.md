#Portfolio Website

A modern, responsive portfolio website built with Vue.js showcasing my skills, experience, and projects.

## Features

- **Responsive Design**: Optimized for all device sizes
- **Modern UI**: Clean, professional design with smooth animations
- **Single Page Application**: Smooth scrolling navigation
- **Interactive Elements**: Hover effects and smooth transitions
- **SEO Optimized**: Semantic HTML structure
- **Fast Loading**: Optimized assets and modern build tools

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **Technical Skills**: Organized grid of programming languages and technologies
3. **Experience**: Timeline format showcasing professional experience
4. **Projects**: Project cards with descriptions and tech stacks
5. **Certifications & Awards**: Credentials and achievements
6. **Contact**: Contact information and social links

## Tech Stack

- **Frontend**: Vue.js 3
- **Build Tool**: Vite
- **Styling**: CSS3 with modern features
- **Icons**: Font Awesome
- **Fonts**: Inter (Google Fonts)
